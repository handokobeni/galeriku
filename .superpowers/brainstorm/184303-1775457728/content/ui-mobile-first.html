Mobile-First UX Decisions

Navigation
Mobile: Bottom tab bar (thumb-friendly)
Desktop: Top nav bar
Upload button: FAB-style di center bottom nav
Touch Gestures
Swipe ←→: navigate media
Pinch: zoom photo
Swipe up: show comments
Pull down: refresh album
Grid Layout
Mobile: 2 columns (album & media)
Tablet: 3 columns
Desktop: 4-5 columns
Masonry-like with varying heights
Media Viewer
Mobile: Full-screen, swipe nav, bottom sheet comments
Desktop: Lightbox overlay, side panel comments
Keyboard nav on desktop (←→ Esc)
Upload
Mobile: Camera/gallery picker, bottom sheet progress
Desktop: Drag & drop zone, modal progress
Both: batch upload with per-file status
Performance
Lazy loading images (IntersectionObserver)
Thumbnail first, full-res on demand
Skeleton loading states
Optimistic UI updates
