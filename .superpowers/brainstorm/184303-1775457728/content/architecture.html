Key Design Decisions

Direct Upload ke R2
File tidak lewat server Vercel — hemat bandwidth & menghindari limit 4.5MB body size
Neon Serverless Driver
HTTP-based connection, cocok untuk Vercel serverless — tidak perlu persistent connection pool
Thumbnail di Client-side
Generate thumbnail di browser sebelum upload — menghindari kebutuhan background job di Vercel
Signed URL untuk Akses
Media diakses via time-limited signed URL — R2 bucket tetap private, akses dikontrol via API
