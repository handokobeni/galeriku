Galeriku — Upload & Media Flow

Presigned URL upload, client-side thumbnail, streaming & download

Upload Flow (Photo & Video)
1
User Pilih File
Drag & drop atau file picker
Validate: type, size
Photo max 20MB
Video max 500MB
→
2
Generate Thumbnail
Client-side
Photo: Canvas API resize
Video: capture frame via <video>
Output: 400px WebP
→
3
Request Presigned URLs
POST /api/upload/presign
Server validates:
• Auth session ✓
• Album permission ✓
• File type & size ✓
Returns 2 presigned URLs
→
4
Upload ke R2
Direct PUT ke R2
Parallel upload:
• Original file → R2
• Thumbnail → R2
Progress bar di UI
→
5
Save Metadata
Server Action: createMedia
Insert ke DB:
• r2_key, thumbnail_r2_key
• mime_type, size
• width, height, duration
Revalidate album page
R2 Key Structure
media/{album_id}/{media_id}/original.{ext}
media/{album_id}/{media_id}/thumb.webp
View — Photo
1. Album page loads → show thumbnails
2. Thumbnails via signed URL (1hr expiry)
3. Click photo → MediaViewer opens
4. Request signed URL for original
5. Display full-res with lazy loading
6. Pinch to zoom, swipe to navigate
Optimization:
• Thumbnail URLs batch-fetched per page
• URLs cached client-side until expiry
• Infinite scroll pagination (20 items)
Stream — Video
1. Click video → MediaViewer opens
2. Request signed URL for video file
3. HTML5 <video> with signed URL src
4. R2 supports Range requests
5. Browser handles seek/buffer natively
Note: No transcoding — R2
serves original file. Browsers handle
MP4/WebM/MOV natively. Signed URL
expiry: 4 hours for video.
Download
1. User clicks download button
2. Request signed URL with
Content-Disposition: attachment
3. Browser triggers native download
4. Original file, original filename
Bulk download:
• Select multiple → download as zip
• Generated client-side via JSZip
• Fetch files via signed URLs → zip
Batch Upload Support
✅ Multiple file selection (file picker + drag & drop)
✅ Queue system — upload 3 files concurrently, rest queued
✅ Per-file progress bar + overall progress
✅ Retry failed uploads (max 3 attempts)
✅ Cancel individual or all uploads
